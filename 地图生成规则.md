# 坦克大战地图生成规则

## 1. 基地规则
- 基地必须位于地图最底部中间位置（默认Y坐标为550，地图高度600）
- 基地类型为5（特殊墙体）
- 基地周围必须被砖墙（类型2）保护：
  - 左侧：基地X坐标 - 50
  - 右侧：基地X坐标 + 50
  - 上方：基地Y坐标 - 50
  - 左上方：基地X坐标 - 50，Y坐标 - 50
  - 右上方：基地X坐标 + 50，Y坐标 - 50

## 2. 玩家出生点规则
- 不能与墙体位置重复
- 玩家1出生点：基地X坐标 - 100，Y坐标 = 550
- 玩家2出生点：基地X坐标 + 100，Y坐标 = 550
- 出生点必须位于地图底部，确保玩家有足够的活动空间

## 3. 敌人出生点规则
- 不能与墙体位置重复
- 敌人出生点必须位于地图最顶部（Y坐标约50左右）
- 至少需要3个敌人出生点，分布在地图顶部不同位置
- 出生点应均匀分布，避免过于集中

## 4. 墙体规则
- 不同类型的墙体：
  - 类型1：砖墙（可破坏）
  - 类型2：钢墙（不可破坏）
  - 类型3：草地（可隐藏坦克）
  - 类型4：河流（不可穿越，除非有特殊道具）
  - 类型5：基地（核心目标）
- 避免墙体位置重复
- 墙体布局应合理，提供多样化的战术选择

## 5. 地图尺寸
- 默认地图尺寸：宽度800，高度600
- 所有坐标必须在此范围内

## 6. 平衡规则
- 确保玩家和敌人都有合理的活动空间
- 避免过于简单或过于复杂的布局
- 提供足够的掩体和战术通道
