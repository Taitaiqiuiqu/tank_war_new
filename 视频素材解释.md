# 视频素材解释

```
videos/
```

## 视频素材说明

### 别怕，还有我呢.mp4
- **触发条件**：双人联机模式时，一个玩家生命数耗尽
- **显示区域**：存活的玩家坦克附近

### 笑.mp4
- **触发条件**：游戏胜利
- **显示区域**：游戏胜利界面

### 诶呀，气死我了.mp4
- **触发条件**：游戏失败
- **显示区域**：游戏失败界面

### 难道他真的是赋能哥.mp4
- **触发条件**：玩家坦克，被敌人坦克击败
- **显示区域**：被击败的玩家坦克位置

### 颗秒.mp4
- **触发条件**：玩家坦克，吃到手榴弹道具
- **显示区域**：覆盖游戏地图
