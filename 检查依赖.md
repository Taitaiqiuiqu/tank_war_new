# 如何检查项目依赖是否已安装

## 方法 1：使用 pip list（最简单）

```bash
# Windows
pip list | findstr /i "pygame moviepy"

# Linux/Mac
pip list | grep -i "pygame\|moviepy"
```

**输出示例：**
```
moviepy         2.2.1
pygame          2.5.2
pygame-gui      0.6.9
```

## 方法 2：使用 pip show（查看详细信息）

```bash
# 检查 pygame
pip show pygame

# 检查 pygame_gui
pip show pygame-gui

# 检查 moviepy
pip show moviepy
```

**输出示例：**
```
Name: pygame
Version: 2.5.2
...
```

## 方法 3：使用 pip freeze（生成依赖列表）

```bash
pip freeze | findstr /i "pygame moviepy"
```

## 方法 4：使用 Python 脚本检查（推荐）

运行项目根目录下的 `check_dependencies.py`：

```bash
python check_dependencies.py
```

这个脚本会：
- 检查所有必需依赖是否安装
- 检查可选依赖是否安装
- 显示每个包的版本号
- 检查 Python 版本是否符合要求

## 方法 5：直接在 Python 中测试导入

```python
# 测试必需依赖
try:
    import pygame
    print(f"pygame 已安装，版本: {pygame.version.ver}")
except ImportError:
    print("pygame 未安装")

try:
    import pygame_gui
    print("pygame_gui 已安装")
except ImportError:
    print("pygame_gui 未安装")

# 测试可选依赖
try:
    import moviepy
    print("moviepy 已安装")
except ImportError:
    print("moviepy 未安装（可选）")
```

## 快速检查命令（一键检查所有依赖）

### Windows PowerShell:
```powershell
@("pygame", "pygame-gui", "moviepy") | ForEach-Object { pip show $_ | Select-String "Name|Version" }
```

### Windows CMD:
```cmd
pip show pygame && pip show pygame-gui && pip show moviepy
```

### Linux/Mac:
```bash
for pkg in pygame pygame-gui moviepy; do pip show $pkg | grep -E "Name|Version"; done
```

## 如果依赖未安装，安装方法：

```bash
# 安装所有依赖
pip install -r requirements.txt

# 或单独安装
pip install pygame==2.5.2
pip install pygame_gui==0.6.9
pip install moviepy
```

## 检查结果说明

- **pygame**: 必需，版本应为 2.5.2
- **pygame_gui**: 必需，版本应为 0.6.9
- **moviepy**: 可选，用于视频播放功能，如果未安装视频功能会使用占位图
