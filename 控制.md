### wasd 控制玩家坦克移动
### 空格 发射子弹
### esc 游戏菜单
### 坦克前 生成对应道具
1. Ctrl + 1：坦克（加命）
- 效果：增加一条生命

2. Ctrl + 2：时钟/秒表（时间静止）
- 效果：规定时间内敌方坦克静止不动，玩家可自由攻击

3. Ctrl + 3：铁锹/铲子（基地防御）
- 效果：一段时间内基地周围红砖变为钢墙，并修复被破坏的红砖

4. Ctrl + 4：手榴弹/炸弹（全屏杀）
- 玩家捡起：全场敌军坦克爆炸
- 敌人捡起：敌人自己爆炸

5. Ctrl + 5：五角星（武器升级）
- 吃一个：增加射击速度，变成两发炮弹，升级到一级
- 吃两个：可消灭铁块，升级到二级
- 吃三个：可消灭草地，升级到三级

6. Ctrl + 6：头盔（无敌）
- 效果：一段时间内坦克周围张开护盾，免疫所有攻击

7. Ctrl + 7：手枪（超级武器）
- 效果：消灭铁块，射速快，升级到二级；连续吃两个可消灭草丛，升级到三级

8. Ctrl + 8：船（水上行走）
- 效果：可过河
